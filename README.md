## 盘中

开盘泛微网络不及预期就直接走，这笔保本；梦网低开不及预期，拉高就走了，后来发现卖飞。如果梦网小高开会多等一等。还有个卖出原因就是昨天提到的浙数文化直接大单一字，所以感觉梦网今天没啥地位了，低开的话了拉高保住受益就跑路，超预期属于认知外了。豪能开盘就走了。买了汉威、双林、北特和移远。都是竞价买了点，开盘又补了点。
